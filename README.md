# Ocean Pulse

Static website for the Ocean Pulse project — a front-end prototype built with HTML, CSS, and static assets.

## Repository

Remote: https://github.com/Menuka-Irindu/Ocean-Pulse.git

## Project structure

- `HTML/` — HTML pages (home, splash, volunteer, validation, page editor, etc.)
- `CSS/` — stylesheets used by the pages
- `icons/`, `images/`, `video/` — static assets

## Local preview

Options to preview the site locally:

- Open `HTML/home.html` (or other files in `HTML/`) directly in a browser.
- Serve the folder with Python 3 (recommended for correct relative paths):

```powershell
cd "C:\Users\User\Downloads\w2120778_G11_Team 6"
python -m http.server 8000
# then open http://localhost:8000/HTML/home.html
```

- Or use the VS Code Live Server extension to run a local server and auto-reload.
